# File Handling

## Mods
- r = for reading if file not exist error
- x = create a file if file exist error
- w = if file exist delete texts and write if not exist create
- a = append new text last of all seek() is not working with that mod
- w+ = same as w but you can read too
- a+ = same as a but you can read too
- r+ = read and overwrite or write
